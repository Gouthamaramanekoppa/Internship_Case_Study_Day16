# Internship_Case_Study_Day16

The XGBoost (Extreme Gradient Boosting) model was implemented to improve fraud detection performance. XGBoost is an advanced ensemble learning algorithm that builds decision trees sequentially and optimizes prediction errors using gradient boosting.
This model is widely used in industry due to its high accuracy, efficiency, and ability to handle complex datasets.

Objectives
The objectives of this phase were:
- Implement the XGBoost classifier
- Train the model on the prepared dataset
- Generate predictions for the test dataset
- Evaluate the model using performance metrics
- Compare its performance with Logistic Regression and Random Forest


Model Training Process
The following steps were carried out:
1. Load the dataset
2. Separate features and target variable
3. Split the dataset into training and testing sets
4. Train the XGBoost model
5. Generate predictions
6. Evaluate model performance

About XGBoost
XGBoost is a powerful gradient boosting algorithm designed for speed and performance.
Key advantages:
- High predictive accuracy
- Handles large datasets efficiently
- Prevents overfitting using regularization
- Supports parallel processing
- Widely used in Kaggle competitions and industry


Evaluation Metrics
The model was evaluated using:
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

For fraud detection systems, **Recall** is critical because it measures the model's ability to correctly identify fraudulent transactions.
